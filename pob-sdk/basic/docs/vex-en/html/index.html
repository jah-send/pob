1       Introduction

 

The goal of this document is to present all the Basic languages functions of the VEX Cortex Microcontroller.

 

To facilitate the use of POB's programming tools, we have conceived our own Basic compiler. Integrated to POB-Tools, it permits to create an embedded program which shares the same format as RISBEE's programs and will be sent on the robot. Several points of interest:

 

- Once the RISBEE firmware is downloaded into the robot, you can indifferently send to it RISBEE or Basic programs; they will be interpreted the same way.

- The memory is more optimal.

- The programs' transfer duration is substantially reduced (as short as for RISBEE programs).

 

From now, you have to install the firmware RISBEE once and for all in the VEX Cortex Microcontroller, and then it will handle all your RISBEE and Basic programs.

 

ß  Hardware requirements

 

You need the following hardware to use the Basic language with the VEX Cortex Micro-Controller:

-       Protobot Robot Kit

-       VEX Cortex Microcontroller

-       VEXnet Joystick & USB Adapter Keys

-       Bumper Switch Kit

-       VEX programming kit

 

ß  Configure the VEX Cortex microcontroller

 

Before programming the VEX Cortex microcontroller with the Basic languages, you need to:

-       Connect your programming kit to your VEX Cortex microcontroller.

-       Connect your Bumper Switch on the digital port 1 of the VEX Cortex microcontroller.

-       Select the COM port between your computer and the VEX Cortex.

-       Install the RISBEE firmware into the VEX Cortex Microcontroller.

 

 

ß  Install firmware

 

To install the RISBEE firmware into the robot, run POB-Tools and go to the ëPreferencesí menu. Next, go to the ëVEX Cortexí tab and click on the ëUpload firmwareí in the Basic part.

Then follow the next steps:

1-    Power on the VEX Cortex Microcontroller.

2-    In POB-TOOLS software, click on the ëDownloadí button.

3-    Press the button on the orange cable to launch the upload process.

4-    At the end, reboot the VEX Cortex Microcontroller: You can now use the Basic languages for the VEX Cortex Microcontroller.

 

Remarks:

†††††† You can found more information to use the VEX Cortex Microcontroller with the RISBEE firmware on the next links:

††††††††††††††† http://www.pob-technology.com/?view=downloads&id=43

 

Warning:

†††††† If you update the POB-TOOLS software, donít forget to update the firmware too!

 

 

ß  How to upload Basic program

 

After you have made your Basic program, you can upload it to the VEX Cortex Microcontroller by clicking on the ëDownloadí button in POB-TOOLS toolbar.

To upload the program, you have to follow this steps :

-       Power on the robot or restart it.

-       Click on the ëDownloadí button in the POB-TOOLS toolbar.

-       Press the button on the VEX programming cable (the orange cable).

-       At the end of process, restart the robot

 

ß  How to run Basic program

 

To run a Basic program in the VEX Cortex Microcontroller, you have to press the bumper switch connected on the digital port 1.

 

Your program launch immediately after releasing the bumper switch.
