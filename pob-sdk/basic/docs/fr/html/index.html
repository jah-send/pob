La tÈlÈcommande Joypad permet díenvoyer des donnÈes sur le rÈseau sans fil Wifi et díÍtre reÁu par le module Wifi du robot.

Pour configurer le module Wifi du robot, il vous faut lire le document dÈdiÈ ‡ ce module Wifi sur le site http://www.pob-technology.com.

 

 

ß  GetJoypad

ParamËtre 1 : Une structure Joypad.

La fonction Ècrit les valeurs du joystick, des sliders et des boutons de la tÈlÈcommande dans la structure Joypad.

 

La structure Joypad contient les champs suivants :

speed1 : Vitesse roue gauche entre -150 et 150.

speed2 : Vitesse roue droite entre -150 et 150.

slider1 : Valeur entre 0 et 180.

slider2 : Valeur entre 0 et 180.

buttonA : Etat du bouton A ( 0 ou 1 )

buttonB : Etat du bouton B ( 0 ou 1 )

buttonC : Etat du bouton C ( 0 ou 1 )

buttonD : Etat du bouton D ( 0 ou 1 )

 

Procedure MainProcedure()

; Create a remoteJoystick structure :

remoteJoystick.Joypad

 

; Create 2 variable and store text :

text1 = "press"

text2 = "notpress"

 

; Main loop :

While 1 = 1

; Get the Joypad value and write it into the remoteJoystick variable :

GetJoypad(remoteJoystick)

 

; Set the DC motor with the Joypad speed :

SetMotor( remoteJoystick\speed1, remoteJoystick\speed2)

 

; Set servomotor 1 with slider1 :

SetServo( 1, remoteJoystick\slider1 )

 

; Check the 'buttonA' of the Joypad :

If remoteJoystick\buttonA = 1

PrintText(text1)

Wait(2)

Else

PrintText(text2)

EndIf

Wend

EndProcedure
