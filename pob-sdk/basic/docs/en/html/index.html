1       Introduction

 

The goal of this document is to present all the POB-Robotics Suite functions of the Basic languages.

 

To facilitate the use of POB's programming tools, we have conceived our own Basic compiler. Integrated to POB-Tools, it permits to create an embedded assembler program which shares the same format as RISBEE's programs and will be sent on the robot. Several points of interest:

 

- Once the risbee firmware is downloaded on the robot, you can indifferently send to it RISBEE or Basic programs; they will be interpreted the same way.

- The memory is more optimal.

- The programs' transfer duration is substantially reduced (as short as for RISBEE programs).

 

From now, you have to install the firmware RISBEE once and for all on the POB Robotics Suite robot, and then it will handle all your RISBEE and Basic programs.

 

ß  Install firmware

 

To install the risbee firmware into the robot, run POB-Tools and go to the ëPreferencesí menu. Next, go to the ëPOB Robotics Suiteí tab and click on the ëUpload firmwareí in the Basic part.

1-  On the robot, power off the robot, then press the programming button and power on.

2-  In pob-tools click on the ëDownloadí button the run the uploading process.

3-  At the end, reboot the robot: It will display ëRISBEEí on the LCD Screen.

6       Joypad remote function

 

The Joypad function must be used with a Wifi module in the robot and the Joypad application for smartphone.

If you want configure the Wifi module of the robot, please read the Wifi configuration document on the web site : http://www.pob-technology.com.

 

 

ß  GetJoypad

 

Get state of the remote Joypad application.

Parameter 1 : A Joypad structure.

The function write the values of joystick, sliders and bouton of the remote Joypad.

 

The Joypad structure has the following fields :

speed1 : Left speed between -150 and 150.

speed2 : Right speed between -150 and 150.

slider1 : Slider value between 0 and 180.

slider2 : Slider value between 0 and 180.

buttonA : button A state ( 0 or 1 )

buttonB : button B state ( 0 or 1 )

buttonC : button C state ( 0 or 1 )

buttonD : button D state ( 0 or 1 )

 

Procedure MainProcedure()

; Create a remoteJoystick structure :

remoteJoystick.Joypad

 

; Create 2 variable and store text :

text1 = "press"

text2 = "notpress"

 

; Main loop :

While 1 = 1

; Get the Joypad value and write it into the remoteJoystick variable :

GetJoypad(remoteJoystick)

 

; Set the DC motor with the Joypad speed :

SetMotor( remoteJoystick\speed1, remoteJoystick\speed2)

 

; Set servomotor 1 with slider1 :

SetServo( 1, remoteJoystick\slider1 )

 

; Check the 'buttonA' of the Joypad :

If remoteJoystick\buttonA = 1

PrintText(text1)

Wait(2)

Else

PrintText(text2)

EndIf

Wend

EndProcedure
