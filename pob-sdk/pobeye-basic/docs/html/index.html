ß        ConfigPortA / ConfigPortD / ConfigPortC

 

Set the way of the digital pins of the POB-PROTO: POB-PROTO has 3 digital input/output ports:

-         port A has 6 input/output, that can be set as input or as output.

-         port C has 6 input/output, that can be set as input or as output or as servomotor output.

-         port D has 8 input/output, that can be set as input or as output.

 

Warning:

-         Only the port C can manage servomotor.

-         port C have the pins 0, 1, 2, 3, 6 et 7 ready to use. The pin 4 and 5 of port C donít exist!

ConfigPortD(6,DIGITALOUTPUT)

ConfigPortD(7,DIGITALOUTPUT)

ConfigPortC(1,ASSERVO)††

; port A, pins 4 and 5 as digital output:
