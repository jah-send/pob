Librairie POB-Robotics-Suite

v1.O

du POB-Robotics-Suite :

Le POB-Robotics-Suite est un puissant robot : L'électronique, la mécanique et le logiciel vous permettent de gérer les capteurs, les actionneurs et réaliser les actions de vos programmes !

Le matériel dispose de :

  • 8 ports pour connecter capteurs et actionneurs.
  • Chaque port fournit alimentation, masse, entrée analogique, entrée/sortie numérique et bus i2c.
  • 8 connecteurs pour servomoteurs standard.
  • 1 connecteur usb (usb vers port série).
  • 2 moteurs à courant continu.
  • Chaque moteur est contrôlé en vitesse et en position par une roue codeuse.
  • 2 microcontrôleurs (chaque microcontrôleur est un ARM7TDMI).

Les microcontrôleurs :

  • Un Microcontrôleur contient le processeur ARM7, la mémoire RAM, la mémoire flash (pour stocker le programme) et les périphériques d'entrées/sorties.
  • Le microcontrôleur gére les moteurs, les capteurs et le comportement de votre programme...
  • Le POB-Robotics-Suite est piloté par 2 microcontrôleur nommé UC1 et UC2.
  • Vous programmez sur UC1 avec les logiciels POB-TOOLS ou RISBEE.
  • Le microcontrôleur UC2 gére le bas niveau : les 2 moteurs cc, les 8 servomotors et les roues codeuses. UC2 est flashé à POB.
  • Chaque microcontrôleur a 8 Koctets de ram et 32 Koctets de flash.

Introduction à la librairie :

La librairie POB-Robotics-Suite fournit des fonctions pour utiliser le robot en langage C :

  • Gestion des capteurs : lumière, distance...,
  • Entrées analogique : fonctions analogiques,
  • Entrées/sorties numérique : fonctions d'entrées/sorties,
  • Communications série : bus I2C and uarts,
  • Moteurs et Roues codeuses : Fonctions moteurs,
  • Gestion des servomoteurs ( Servomoteurs),
  • Mémoire Eeprom : fonctions eeprom,
  • Ecran LCD : Fonctions Lcd,
  • Communication sans-fil : réseau Wifi,
  • et fonctions diverses : Fonctions de temps, utilitaires ...

Utiliser la librairie :

Vous devez installer POBTOOLS et le compilateur gcc yagarto. Les deux logiciels sont fournis sur le cdrom livré avec le robot.

Vous pouvez copier/coller le code suivant pour tester votre installation :

 #include <pobRoboticSuite.h>

 int main(void)
 {
    // Initialisation du microcontrôleur du robot :
    InitCpu();

    // Initialisation de l'UART 0 avec le débit 115200 :
    InitUART0(115200);

    // Initialisation du robot et des capteurs :
    InitRobot();

    SendString("Hello world from POB-Robotics-Suite!");

    while(1) ;

    return 0;
 }
