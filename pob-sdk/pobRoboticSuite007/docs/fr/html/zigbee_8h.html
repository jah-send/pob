Description détaillée

Fonction module ZigBee.

Le module ZigBee permet d'avoir un réseau sans fil radio pour le robot.

Sur le robot, le module ZigBee est connecté sur le port série UART 1 (Ce module est optionnel !). Le module ZigBee peut fonctionner en 2 modes :

  • mode Transparent, vous utilisez le ZigBee comme un lien série avec les fonctions UART 1.
  • mode API, vous utilisez le ZigBee en mode paquet : vous pouvez utilisez les fonctions GetBufferFromZigbee et SendBufferToZigbee.

UInt8 GetBufferFromZigbee ( UInt8 *  buffer,
UInt32 *  length,
UInt16 *  address 
)

Récupérer un tableau d'octet depuis le module ZigBee.

Utiliser cette fonction uniquement si vous êtes en mode API ! Si vous avez configurer le module en mode Transparent, il faut utiliser les fonctions UART 1.

Paramètres:
buffer Pointeur sur un tableau où stocker les octets reçus (la taille minimale du tableau doit être votre taille + 16 octets).
[out] length Nombre d'octets reçus.
[out] address Adresse de l'émetteur.
Renvoie:
ZIGBEE_OK si aucune erreur, sinon code d'erreur.
Exemple pour récupérer un tableau du module ZigBee.
 #include <pobRoboticSuite.h>

 int main(void)
 {
    UInt8 result;
    UInt32 size;
    UInt16 addr;
    UInt8 data[2+16];

    // initialisation du cpu et du robot :
    InitCpu();
    InitRobot();

    // initialisation du port série uart 0 :
    InitUART0(115200);

    // initialisation du module zigbee en mode API avec l'adresse 1 :
    InitZigbee(115200,ZIGBEE_MODE_API,1);

    while(1)
    {
        result = GetBufferFromZigbee(data,&size,&addr);

        if(result==ZIGBEE_OK)
        {
            DebugPrintf("Receiver Address: 0x%x Length: %d\r\n", addr, size);
        }

    }

    return 0;
 }
