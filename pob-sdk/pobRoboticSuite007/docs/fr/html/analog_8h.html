UInt16 GetAnalog (UInt8 port)
 Récupère la valeur analogique depuis un port du robot.

Description détaillée

Fonctions analogiques.

Le POB-Robotics-Suite a 8 ports analogiques. Ses ports analogiques mesurent une tension compris entre 0 et 3.3 Volts et la convertit en une valeur sur 10 bits.

Si vous souhaitez utiliser une entrée/sortie numérique, allez voir fonctions d'entrées/sorties.

Example d'entrée analogique
 #include <pobRoboticSuite.h>

 int main(void)
 {
        UInt16 result;

        // Initialise le CPU:
        InitCpu();

        // Recherche et initialise les périphériques connectés sur les ports du POB-Robotics-Suite :
        InitRobot();

        // Lit la valeur analogique du port 2 du robot :
        result = GetAnalog(2);

        return 0;
 }
